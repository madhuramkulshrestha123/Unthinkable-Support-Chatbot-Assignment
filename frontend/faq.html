Frequently Asked Questions

How to use the chatbot?
Select a customer from the customer list, then start chatting with the AI support assistant. The chatbot will provide personalized responses based on the customer's order history.
Is it secure?
Yes, all communications are secured with HTTPS encryption. Customer data is stored securely and is only used to provide personalized support experiences.
What tech stacks does it use?
Frontend: HTML, CSS, JavaScript. Backend: Node.js with Express. Database: In-memory storage (can be extended to MongoDB). AI: Google's Gemini API for natural language processing.
What LLM API does it use?
The chatbot uses Google's Gemini API for generating intelligent responses. It provides contextual answers based on customer history and query content.
Can I create new customers?
Yes, you can create new customers using the "Create Customer" page. Add customer details and their order history to personalize their support experience.
How does the AI personalize responses?
The AI analyzes the customer's order history, purchase patterns, and previous interactions to provide tailored support responses for each individual.
What payment methods are supported?
The system supports Credit Card, Debit Card, UPI, Cash on Delivery, and Net Banking payment methods for customer transactions.
How do I end a chat session?
Click the "Exit" button in the chat window header to end the current chat session and return to the customer selection screen.
